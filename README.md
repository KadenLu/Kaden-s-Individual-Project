# Kaden-s-Individual-Project
For a JEI project. The code should be directly run in a Colab environment ("Open in Colab").
The Colab generates the results for my project, which uses machine learning to test a hypothesis
that demographic featurs are not important to decide anime user archetypes.

To reproduce the results, a user needs to download the MyAnimeList data set from Kaggle, and upload
the animelists_cleaned.csv, anime_cleaned.csv, and user_cleaned.csv files to a corresponding google
drive folder. E.g., my files were stored at "/content/drive/MyDrive/kaden_projects/". The main cells
of the Colab were labelled, and it should be intuitive to understand what each cell does.
